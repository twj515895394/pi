# 0013：Capstone Project

## 本课目标

把前面课程学到的 Pi 设计转化为一个可展示的个人项目或架构方案。

## 产出方向

1. Pi extension：工具审计或计划模式。
2. Java Agent Harness 设计文档。
3. Dify + 自研 Harness 混合架构方案。
4. Coding Agent 企业落地 Demo。

## 核心问题

- 这个项目证明了哪些 Agent Harness 能力？
- 它和单纯使用 Dify 有什么不同？
- 哪些部分可运行，哪些部分是设计方案？

# 课程设计目录

这里存放课程草案。课程草案用于规划下一节课要讲什么、读哪些源码、做什么练习。

正式课程材料后续放入 `../lessons/`，并优先使用 HTML，便于长期复习。

## 命名规则

```text
0001-pi-agent-loop.md
0002-agent-message-vs-llm-message.md
0003-tool-call-execution.md
```

## 草案结构

每个课程草案应包含：

```text
# 课题

## 本课目标
## 前置知识
## 源码范围
## 核心概念
## 源码阅读路径
## 检查问题
## 小练习
## 复盘产出
```

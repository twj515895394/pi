# Coding Tools 风险矩阵

> 状态：待课程 0008 完成后补全。

## 初始维度

| 工具 | 类型 | 主要价值 | 主要风险 | 企业治理建议 |
|---|---|---|---|---|
| read | 只读 | 获取文件内容 | 泄露敏感文件 | 路径白名单 / 敏感路径保护 |
| grep | 只读 | 搜索代码 | 泄露敏感内容 | 路径白名单 / 输出截断 |
| find | 只读 | 查找文件 | 目录探测 | 输出限制 |
| ls | 只读 | 查看结构 | 目录探测 | 输出限制 |
| bash | 执行 | 运行命令 / 测试 / 构建 | 破坏文件 / 网络 / 凭证泄露 | 高风险审批 / 沙箱 |
| edit | 写入 | 精准修改 | 错误改动 | diff 审核 / 文件保护 |
| write | 写入 | 创建或覆盖文件 | 覆盖重要文件 | 文件保护 / 确认 |

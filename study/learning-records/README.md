# 学习记录目录

这里记录已经被验证的关键理解。学习记录不是日报，也不是阅读进度。

## 什么时候写

只有在以下情况才写：

- 已经能用自己的话解释一个关键概念。
- 已经完成练习并通过复盘。
- 已经纠正一个重要误解。
- 学习使命发生变化。

## 命名规则

```text
0001-agent-loop-is-model-tool-observation-cycle.md
0002-tool-is-governed-execution-unit.md
0003-harness-is-runtime-control-layer.md
```

## 最小格式

```md
# 标题

1-3 句话说明学到了什么，以及为什么它会影响后续学习。
```

# Handoffs

本目录记录每节课结束后的交接文档。

## 目的

让学习过程可以跨会话持续推进。任何新的 Agent 或新会话只要读取 `study/CURRENT.md` 和最新 handoff，就能知道：

- 当前学到哪里。
- 已经理解什么。
- 仍然不清楚什么。
- 下节课应该讲什么。
- 哪些文档需要继续更新。

## 命名规则

```text
0000-initial-study-setup.md
0001-pi-agent-loop.md
0002-agent-message-vs-llm-message.md
```

## 规则

- 每节课结束后必须写 handoff。
- Handoff 可以记录过程、反馈和下一步。
- Learning Record 只记录已经被验证的关键理解，不替代 handoff。
